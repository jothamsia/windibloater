# windibloater
Not full but removes most useless apps that come with a new windows install
Debloats windows after a clean install or a major update. 
Very simple works with winget windows official package manager.
How to use:
1. click on code than click on download zip, 
2. Extract the downloaded zip file open the folder and find a file named windowsdebloater.bat, before you run it open microsoft store search for app installer and update it, once done run windowsdebloater.bat as adminstrator.
